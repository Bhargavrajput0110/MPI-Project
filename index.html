SYSTEM REQUISITES
MCU-01

ESP32 Microcontroller

High-performance MCU with integrated Wi-Fi and Dual-mode Bluetooth for core processing and cloud gateway duties.

SEN-01

ZMPT101B Voltage Sensor

Precision active voltage transformer for measuring high-accuracy AC mains voltage for power calculations.

SEN-02

ACS712 Current Sensor

Linear Hall-effect based current sensor designed to measure AC/DC current for real-time load analysis.

SEN-03

DHT11 (Temp & Humidity)

Digital signal acquisition module for environmental data, aiding in correlating heat dissipation with energy load.

MOD-01

ESP32 Expansion Board

Shield providing easy GPIO access and terminal block connections for robust hardware integration.

ACT-01

Relay Module

Controlled electromagnetic switch for isolating the high-voltage load circuit from the logic circuit.

INT-01

Programming Interface

USB connectivity for firmware deployment and hardware debugging via serial communication protocols.
